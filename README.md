# SatNOGS Ground Station Documentation

This repository contains the documentation for our SatNOGS ground station, including hardware, software installation, configuration, testing, and troubleshooting.

## Purpose

The goal of this project is to build, configure, and operate a SatNOGS ground station capable of receiving and tracking satellite signals automatically.

## Current Setup

- Raspberry Pi 4
- SatNOGS ground station + HamLib 
- SatNOGS Rotator
- RTL-SDR V4
- LNA
- RF splitter/combiner
- Cross-Yagi antenna
- Coaxial RF chain (double shielded) + ferites chokes

## Repository Structure
